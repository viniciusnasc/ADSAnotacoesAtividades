Atributos para o body:

bgcolor="[color]" => Define a cor do background

background="[link da imagem]" => Define uma imagem como plano de fundo

link="[color]" => Define a cor de links não visitados

vlink="[color]" => Define a cor de links já visitados

alink="[color]" => Define a cor do hiperlink ao ser pressionado

Atributos para a tag a:

target="_blank" => Ele abre uma janela para ir ao link especificado
